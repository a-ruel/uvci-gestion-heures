# Rapport final du projet – Gestion des heures d'enseignement (UVCI)

## Objectifs atteints
- Authentification sécurisée avec rôles
- Gestion complète des enseignants, cours, séquences, ressources
- Saisie et validation des activités pédagogiques (conception / mise à jour)
- Calcul automatique des volumes horaires selon coefficients de complexité
- Détection des heures complémentaires
- Tableaux de bord personnalisés
- Génération d’états (PDF, Excel) pour les trois profils
- Journalisation des actions (logs)
- Sauvegarde de la base de données
- Graphiques statistiques (admin)

## Contraintes techniques respectées
- Application web responsive
- Base MySQL (InnoDB)
- Gestion des rôles et permissions
- Sauvegarde automatique (manuelle)
